# Sales Performance Analysis (Excel)

## Objective
To analyze sales data and identify monthly trends, top-performing products, and region-wise performance using Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Key Analysis
- Monthly sales trend analysis
- Region-wise and category-wise sales performance
- Top 10 best-selling products

## Outcome
Built an interactive Excel dashboard to support data-driven business decisions.
